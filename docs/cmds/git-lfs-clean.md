# git-lfs-clean

## Name

`git-lfs-clean` — Run the clean filter: read content on stdin, write a pointer on stdout

## Synopsis

```
git-lfs-clean [PATH]
```

## Description

Run the clean filter: read content on stdin, write a pointer on stdout

## Options

### Arguments

- `<PATH>`
    Working-tree path of the file being cleaned (currently unused)
